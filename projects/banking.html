Modern Banking System

A secure and feature-rich banking application developed using PHP, MySQL, and JavaScript.

Overview

This project simulates a real-world banking environment, designed for users and administrators to manage accounts securely. It supports login authentication, transactions, user role management, and responsive interfaces.

Key Features

  • 🔐 Secure login & password hashing
  • 📜 PDF receipts using DomPDF
  • 💼 Admin dashboard with analytics
  • 💳 Deposit, Withdraw, and Balance Check
  • 📥 Fund transfer between users
  • 🔍 Filter transactions by date, type & amount
  • 📧 Email alerts via PHPMailer API
  • 📱 Mobile-responsive layout
  • 📈 Charts & graphs using Chart.js
